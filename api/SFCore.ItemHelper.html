Class ItemHelper

Namespace
SFCore
Assembly
SFCore.dll

Item helper class for easily adding custom items. The mod using this needs to handle the following:

  • 0 to 3 PlayerData bools per item
  • 0 to 1 PlayerData int per item
  • 1 to 3 name language strings per item
  • 1 to 4 description language strings per item
